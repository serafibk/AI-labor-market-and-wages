# AI-labor-market-and-wages

This project sets up a simulation to investigate the possible effects of the presence of a salary prediction tool in a simple labor market compared to a market 1) with no pay transparency and 2) pay transparency via firm-posted salary ranges instead. 

The simulation is set up in ```labor_market_q_learning_simulation.py``` and the outputs of the simulation experiments are analyzed in ```analyze_simulation_outputs.py```.
